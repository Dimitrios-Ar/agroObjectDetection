calculate_mAP: calculates the mean Average precision of the detector

detect_image: makes detection on an image

detect_multiple_images: makes detection on multiple images 

detect_to_enlarge_dataset: detects on new images to be later used as training data

detect_video: makes predictions on a video (on each frame)

train: trains a new model

train_consecutively: trains more models one after the other and saves the files between

For the above, specific configuration files are required. These are explained into the code/data/ folder.
